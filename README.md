# ticketfrontend
Nuxtjs frontend for ticket System

To test: 
  1.npm install
  2.npm run dev
  3.Go to http://localhost:3000/
This project uses NuxtJS, combined with NodeJS for backend and MongoDB cloud for Database.

After clicking into http://localhost:3000/.
